Create Command Plugin Improve page

  1. Install the Pingo.Commandline.PluginTemplate Extension
    • direct: Extension
    • Visual Studio 2013: Tools -> Extensions and Updates… -> Online
    • restart visual studio 2013
  2. Create a new Project
    • Use the newly installed Pingo.Commandline.PluginTemplate
    • Name it “My.Sweet.Command”
    • A wizard prompt asks for what your command name should be.
      • Enter “Sweet”
    • Press “Done”
    • Build “My.Sweet.Command”
    • Add the “My.Sweet.Command” project as a reference to “ConsoleMe”
    • Build “ConsoleMe”
  3. Done and Run!
    You should see your “Sweet” command in the list of now usable commands.
  4. Modify your command project to suit.
  5. You can now uninstall the Pingo.JsonConverters.Commands and Pingo.TestCommands nugets from “ConsoleMe”, Unless you really want to keep them in there for their profound awesomeness.
